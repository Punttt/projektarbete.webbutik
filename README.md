Direktfärd – Prototyp

Detta repo innehåller en prototyp för tjänsten Direktfärd, ett fiktivt färdtjänstsystem utvecklat som kursuppgift. Syftet är att visa hur digitala lösningar kan utformas med fokus på tillgänglighet, enkelhet och trygghet för användare med olika behov.
Projektet hör till kursen Webbanvändbarhet (DT068G), HT2025.

Innehåll
- Förarbete med målgruppsanalys, personas och scenarier
- Prototyp med startsida, bokningsflöde och bekräftelse
- Footer med länkar till Om oss, Tillgänglighetsredogörelse och Integritetspolicy
- Sidor för policies och företagsinformation
